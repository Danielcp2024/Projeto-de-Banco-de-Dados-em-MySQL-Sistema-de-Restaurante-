# Projeto de Banco de Dados em MySQL – Sistema de Restaurante

Projeto de modelagem e manipulação de banco de dados relacional em MySQL, simulando um sistema de restaurante. O banco foi estruturado para armazenar e organizar informações de clientes, produtos e pedidos, permitindo consultas e manipulação eficiente dos dados.

## Tecnologias Utilizadas

MySQL

SQL

## Funcionalidades Implementadas

Criação e estruturação de tabelas relacionais

Inserção de dados com comandos INSERT

Consulta de dados com SELECT

Organização de registros por entidades

Simulação de um sistema comercial real

## Objetivo do Projeto

Demonstrar conhecimentos em banco de dados relacional, linguagem SQL e organização de informações para suporte a sistemas de gestão.
